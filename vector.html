Vector

Learn about vector concepts, operations and solved examples ✅

Overview

Definition: A vector is a quantity that has both magnitude and direction. Represented as an arrow or in component form.

Types of Vectors:

1. Zero Vector: Vector with magnitude 0
Example: 0

2. Unit Vector: Vector with magnitude 1 along x, y, z axes
Example: î, ĵ, k̂

3. Equal Vectors: Same magnitude and direction
Example: A = B

4. Negative Vectors: Opposite direction
Example: If A = 3i + 4j, then -A = -3i - 4j

5. Position Vector: Represents position of a point relative to origin
Example: P(x,y,z) → r = xî + yĵ + zk̂

Vector Operations:

• Addition: A + B = (ax+bx, ay+by, az+bz)
• Subtraction: A - B = (ax-bx, ay-by, az-bz)
• Scalar Multiplication: kA = (kax, kay, kaz)
• Dot Product: A · B = axbx + ayby + azbz = |A||B|cosθ
• Cross Product: |A × B| = |A||B|sinθ; direction by right-hand rule
• Magnitude: |A| = √(ax² + ay² + az²)
• Unit Vector: Â = A/|A|

Examples

Example 1: Vector Addition

A = 2i + 3j, B = i + 4j → A + B = 3i + 7j

Example 2: Vector Subtraction

A = 5i + 2j, B = 3i + 7j → A - B = 2i - 5j

Example 3: Dot Product

A = 2i + 3j, B = i + 4j → A · B = 2 + 12 = 14

Example 4: Cross Product

A = i + 2j, B = 3i + j → A × B = -5k

Example 5: Magnitude & Unit Vector

A = 3i + 4j → |A| = 5, Unit vector = (3/5)i + (4/5)j
