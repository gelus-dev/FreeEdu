Sets

Learn about sets, types, operations, and solved examples ✅

Overview

Definition: A set is a well-defined collection of distinct objects, considered as an object in its own right. The objects are called elements or members of the set.

Types of Sets:

1. Empty Set (Null Set): A set with no elements.
Example: ∅ or { }

2. Finite Set: A set with a countable number of elements.
Example: {1, 2, 3, 4}

3. Infinite Set: A set with infinite elements.
Example: {1, 2, 3, …}

4. Equal Sets: Sets with exactly the same elements.
Example: A = {1,2,3}, B = {3,2,1} → A = B

5. Subset: A set A is a subset of B if every element of A is also in B.
Example: A = {1,2}, B = {1,2,3} → A ⊆ B

6. Proper Subset: A ⊂ B if A ⊆ B and A ≠ B.

7. Universal Set: The set containing all possible elements under consideration.
Example: U = {1,2,3,4,5,6}

8. Complement of a Set: Elements in the universal set not in A.
Example: U = {1,2,3,4,5}, A = {1,2} → A' = {3,4,5}

Set Operations:

• Union (A ∪ B): Set containing elements of A or B or both.
• Intersection (A ∩ B): Set containing elements common to A and B.
• Difference (A - B): Elements in A but not in B.
• Complement (A'): Elements in U but not in A.
• Cartesian Product (A × B): Set of all ordered pairs (a,b) where a ∈ A, b ∈ B.

Examples

Example 1: Union of Sets

Given: A = {1,2,3}, B = {3,4,5}
Find: A ∪ B
Solution: A ∪ B = {1,2,3,4,5}

Example 2: Intersection of Sets

Given: A = {1,2,3}, B = {3,4,5}
Find: A ∩ B
Solution: A ∩ B = {3}

Example 3: Difference of Sets

Given: A = {1,2,3,4}, B = {3,4,5}
Find: A - B
Solution: A - B = {1,2}

Example 4: Complement of a Set

Given: U = {1,2,3,4,5}, A = {1,2}
Find: A'
Solution: A' = {3,4,5}

Example 5: Cartesian Product

Given: A = {1,2}, B = {x,y}
Find: A × B
Solution: A × B = {(1,x),(1,y),(2,x),(2,y)}

Practice Questions (MCQ)

Question 1:

Which of the following sets is a subset of {1,2,3}?
A) {1,2,3,4}
B) {1,2}
C) {0,1}
D) {4,5}

Question 2:

Let A = {1,2,3}, B = {2,3,4}. Find A ∪ B.
A) {1,2,3}
B) {2,3,4}
C) {1,2,3,4}
D) {1,4}

Question 3:

Let A = {1,2,3}, B = {2,3,4}. Find A ∩ B.
A) {1}
B) {2,3}
C) {3,4}
D) {1,4}

Question 4:

Let U = {1,2,3,4,5}, A = {1,3}. Find A'.
A) {1,3}
B) {2,4,5}
C) {1,2,3}
D) {4,5}

Question 5:

Find the Cartesian product of A = {1,2} and B = {x,y}.
A) {(1,x),(2,y)}
B) {(1,x),(1,y),(2,x),(2,y)}
C) {(x,1),(y,2)}
D) {(1,2),(x,y)}

Question 6:

Which of the following is a proper subset of {1,2,3}?
A) {1,2,3}
B) {1,2}
C) {1,4}
D) {3,4,5}

Question 7:

Let A = {1,2}, B = {2,3}. Find A - B.
A) {1,2}
B) {1}
C) {2}
D) {2,3}

Question 8:

Find (A ∪ B) - (A ∩ B) if A = {1,2,3}, B = {2,3,4}.
A) {2,3}
B) {1,4}
C) {1,2,3,4}
D) {1,3,4}

Question 9:

If A = {1,2,3} and B = {2,3,4}, which of the following is true?
A) A ⊆ B
B) B ⊆ A
C) A ∩ B = {2,3}
D) A ∪ B = {1,2,3}

Question 10:

Which set operation results in elements common to both sets?
A) Union
B) Intersection
C) Difference
D) Complement
