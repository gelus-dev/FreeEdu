Limit and Continuity

Learn about Limit and Continuity concepts, operations and solved examples ✅

Overview

Definition: The limit of a function at a point is the value that the function approaches as the input approaches that point. Continuity means the function has no breaks, jumps, or holes at a given point.

Types of Limits:

1. Finite Limit: Limit exists and is finite.
Example: lim x→2 (x²) = 4

2. Infinite Limit: Limit approaches infinity.
Example: lim x→0 (1/x²) = ∞

3. One-Sided Limit: Limit approached from left or right.
Example: lim x→0⁺ (1/x) = ∞, lim x→0⁻ (1/x) = -∞

4. Limits at Infinity: Limit as x → ∞ or -∞.
Example: lim x→∞ (1/x) = 0

Continuity:

• A function f(x) is continuous at x = a if:
1) f(a) is defined
2) lim x→a f(x) exists
3) lim x→a f(x) = f(a)

Types of Discontinuity:

1. Removable: Limit exists but f(a) ≠ limit.
2. Jump: Left-hand and right-hand limits exist but are not equal.
3. Infinite: Limit approaches infinity.

Key Formulas:

• lim x→0 (sinx)/x = 1
• lim x→0 (1-cosx)/x = 0
• lim x→∞ (1 + 1/x)^x = e
• lim x→0 (1+x)^k = 1

Examples

Example 1: Basic Limit

Given: lim x→3 (2x + 5)
Solution: Substitute x=3 → 2*3+5=11

Example 2: Limit Using Factorization

Given: lim x→2 (x² - 4)/(x - 2)
Solution: Factor numerator: (x-2)(x+2)/(x-2) → x+2 → Substitute x=2 → 4

Example 3: Trigonometric Limit

Given: lim x→0 (sin3x)/x
Solution: lim x→0 (sin3x)/x = lim x→0 (3sin3x)/(3x) = 3*1 = 3

Example 4: One-Sided Limit

Given: f(x) = { x², x<0; x+1, x≥0 }, find lim x→0⁻ f(x) and lim x→0⁺ f(x)
Solution: Left-hand: 0²=0; Right-hand: 0+1=1 → Limit does not exist at x=0

Example 5: Continuity Check

Given: f(x) = { x², x≠1; 2, x=1 }, check continuity at x=1
Solution: lim x→1 f(x) = 1 ≠ f(1)=2 → Discontinuous (removable)

Example 6: Limit at Infinity

Given: lim x→∞ (5x² + 3)/(2x² + 7)
Solution: Divide numerator and denominator by x² → 5/2
