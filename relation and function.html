Relations and Functions

Learn about Relations and Functions with solved examples and Questions✅

Overview

Definition: A relation is a set of ordered pairs. A function is a special type of relation in which each element of the domain corresponds to exactly one element in the codomain.

Types of Relations:

1. Reflexive Relation: A relation R on set A is reflexive if (a, a) ∈ R for all a ∈ A.
Example: On set {1,2,3}, R = {(1,1),(2,2),(3,3),(1,2)} is reflexive.

2. Symmetric Relation: R is symmetric if (a,b) ∈ R ⇒ (b,a) ∈ R.
Example: R = {(1,2),(2,1),(2,3),(3,2)}

3. Transitive Relation: R is transitive if (a,b) ∈ R and (b,c) ∈ R ⇒ (a,c) ∈ R.
Example: R = {(1,2),(2,3),(1,3)}

4. Equivalence Relation: R is reflexive, symmetric, and transitive.
Example: Congruence modulo n.

Types of Functions:

1. One-to-One (Injective): Each element of domain maps to a unique element of codomain.
Example: f(x) = 2x

2. Onto (Surjective): Every element of codomain has a pre-image in domain.
Example: f(x) = x³

3. Bijective: Function that is both injective and surjective.
Example: f(x) = x+1 from Z → Z

Important Formulas and Concepts:

• Domain: Set of all first elements of ordered pairs.
• Codomain: Set of possible second elements.
• Range: Set of all actual second elements.
• Composition: (f ∘ g)(x) = f(g(x))
• Inverse: f⁻¹(y) = x if f(x) = y
• Number of functions from set A (m elements) to set B (n elements) = n^m

Examples

Example 1: Determine Function

Given: R = {(1,2),(2,3),(3,4)}
Check: Is R a function?
Solution: Each element of domain maps to exactly one element → Yes, function.

Example 2: One-to-One Function

Given: f(x) = 2x + 1
Check: Is f one-to-one?
Solution: f(x₁) = f(x₂) ⇒ 2x₁+1 = 2x₂+1 ⇒ x₁ = x₂ → Injective → Yes

Example 3: Onto Function

Given: f(x) = x², f: R → R
Check: Is f onto?
Solution: Not all real numbers are squares (e.g., -1) → Not onto

Example 4: Composition

Given: f(x) = x+1, g(x) = 2x
Find: (f ∘ g)(x)
Solution: f(g(x)) = f(2x) = 2x + 1

Example 5: Inverse Function

Given: f(x) = 3x - 2
Find: f⁻¹(x)
Solution: y = 3x - 2 → x = (y+2)/3 → f⁻¹(x) = (x+2)/3

Example 6: Number of Functions

Given: Set A = {1,2,3}, B = {a,b}
Find: Number of functions from A to B
Solution: n^m = 2^3 = 8

Practice Questions (Advanced MCQ)

Question 1:

Which of the following relations on set {1,2,3} is an equivalence relation?
A) R1 = {(1,1),(2,2),(3,3),(1,2)}
B) R2 = {(1,1),(2,2),(3,3),(1,2),(2,1)}
C) R3 = {(1,1),(2,2),(3,3),(1,2),(2,1),(2,3),(3,2),(1,3),(3,1)}
D) R4 = {(1,2),(2,3)}

Question 2:

f: R → R, f(x) = 2x + 3. Which property holds?
A) Injective only
B) Surjective only
C) Bijective
D) Neither

Question 3:

f(x) = x², g(x) = √x, x ≥ 0. Find (g ∘ f)(4)
A) 2
B) 4
C) 16
D) 8

Question 4:

Inverse of f(x) = 5x - 7 is
A) (x+7)/5
B) (x-7)/5
C) 5x +7
D) (5-x)/7

Question 5:

Number of functions from set A = {1,2,3,4} to B = {a,b} is
A) 4
B) 8
C) 16
D) 12

Question 6:

f: R → R, f(x) = x³. Which statement is correct?
A) One-to-one only
B) Onto only
C) Bijective
D) Neither

Question 7:

Which relation is symmetric but not transitive on {1,2,3}?
A) {(1,1),(2,2),(3,3)}
B) {(1,2),(2,1),(2,3),(3,2)}
C) {(1,2),(2,1),(1,1),(2,2)}
D) {(1,2),(2,1),(1,3)}

Question 8:

Domain of f(x) = √(x-1) is
A) x ≥ 0
B) x > 0
C) x ≥ 1
D) x > 1

Question 9:

If f(x) = 2x+3 and g(x) = x², find (f ∘ g)(2)
A) 7
B) 8
C) 11
D) 10

Question 10:

Which function is not one-to-one?
A) f(x) = 3x + 1
B) f(x) = x², x ∈ R
C) f(x) = x³
D) f(x) = 2x - 5

Answer Key

Answer 1:

C) R3; reflexive, symmetric, and transitive

Answer 2:

C) Bijective; linear function maps R → R one-to-one and onto

Answer 3:

A) 2; g(f(4)) = g(16) = √16 = 4

Answer 4:

B) (x+7)/5

Answer 5:

C) 16; number of functions = 2⁴ = 16

Answer 6:

C) Bijective; f(x)=x³ is both one-to-one and onto R → R

Answer 7:

D) {(1,2),(2,1),(1,3)}; symmetric but not transitive

Answer 8:

C) x ≥ 1; inside square root must be ≥ 0

Answer 9:

C) 11; f(g(2)) = f(4) = 2*4+3=11

Answer 10:

B) f(x)=x²; fails one-to-one over R
